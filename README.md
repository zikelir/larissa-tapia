# larissa-tapia

## Build Setup

```bash
# install dependencies
$ yarn install
#or
$ npm install

# serve with hot reload at localhost:3000
$ yarn dev
#or
$ npm run dev

# build for production and launch server
$ yarn build
$ yarn start
#or
$ npm run build
$ npm start

# generate static project
$ yarn generate
# or
$ npm run generate
```
